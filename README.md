# 🎵 musical-octo-winner

A modern music player application with intuitive controls and support for multiple audio formats.

## Features

- 🎵 **Multi-format Support**: Play MP3, WAV, FLAC, and more
- 🎨 **Intuitive UI**: Clean and user-friendly interface
- ▶️ **Playback Control**: Play, pause, skip, and volume control
- 📋 **Playlist Management**: Create and manage playlists
- 🔊 **Audio Visualization**: Real-time audio waveform display
- 🎯 **Search**: Quick search through your music library

## Installation

### Prerequisites
- Python 3.8 or higher
- pip package manager

### Setup

```bash
# Clone the repository
git clone https://github.com/ramirezrolando222222-eng/musical-octo-winner.git
cd musical-octo-winner

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py
```

## Usage

```bash
python main.py
```

### Controls
- **Play**: Click the play button or press spacebar
- **Pause**: Click pause or press spacebar again
- **Next Track**: Click next or press 'N'
- **Previous Track**: Click previous or press 'P'
- **Volume**: Adjust with the volume slider
